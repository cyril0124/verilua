# 类型概览

本文档汇总了 `Verilua` 中所有面向用户的**可实例化类型**，按功能模块分组，并标注文档覆盖状态。

- **已文档**的类型可直接点击链接查看详细 API。
- **Coming soon** 的类型尚未提供独立文档页面，用户可通过源码中的 LuaCATS 注解了解详情。
